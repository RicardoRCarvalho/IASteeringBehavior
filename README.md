# IASteeringBehavior
Carro azul: obstaculo
Carro amarelo: Pathfinding: vai para uma coordenada após chegar na outra.
ao total são quatro coordenadas
Carro verde: Movimento aleatorio + flee: Adiciona velocidade em direção aleatoria em volta do carro
também é adicionada força de maneira que o carro fuge do mouse.
Carro vermelho: Seek + arrive + avoid: Ao clickar carro vai para posição clicada.
após determinada distancia do ponto, é aplicada uma força contratia a sua direção.
Ao chegar perto do obstaculo(carro azul) é aplicada uma força para se afastar dele
