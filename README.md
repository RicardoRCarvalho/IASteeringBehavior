# IASteeringBehavior
Carro azul: obstaculo    
Carro amarelo: Pathfinding: vai para uma coordenada após chegar na outra.    
ao total são quatro coordenadas    
Carro verde: Movimento aleatorio + flee: Adiciona velocidade em direção aleatoria dentro de um "ciruclo" na frente do carro    
também é adicionada força de maneira que o carro fuge do mouse.    
Carro vermelho: Seek + arrive + avoid: Ao clicar adiciona força na direção da posição clicada.       
após determinada distancia do ponto, é aplicada uma força contraria a sua direção para que ele pare.    
Ao chegar perto do obstaculo(carro azul) é aplicada uma força para se afastar dele
